Development Roadmap

Phase 1: MVP

Weeks 1-4
  • Match-3 engine (Phaser 3 + TypeScript)
  • 30 hand-crafted levels with difficulty curve
  • Jest SDK integration (player, data, payments)
  • 3 IAP products (moves, booster, lives)
  • Guest mode support for first 10 levels
  • Submit for Jest review

Phase 2: Social & Retention

Weeks 5-8
  • PvP challenge system via referral links
  • SMS notification scheduling (D0-D7)
  • Daily challenge mode
  • 7-day login streak rewards
  • Leaderboards (weekly/all-time)
  • Gift a friend feature

Phase 3: Scale & Monetize

Weeks 9-12
  • Season Pass (4-week premium track, $5)
  • Level editor for rapid content creation
  • 10+ new levels per week pipeline
  • Seasonal themes and cosmetics
  • Apply for Jest Games Fund ($200K)
  • Analytics-driven difficulty tuning

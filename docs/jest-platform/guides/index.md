# Guides

## Build better games on Jest

This section contains practical guides and best practices for building, launching, and growing games on the Jest platform. Unlike the SDK reference, these guides focus on how to use Jest effectively - covering strategy, design decisions, and platform-specific considerations that help your game perform better over time.

Topics include:
- **Notifications** - How to design effective messaging strategies that drive retention.
- **User acquisition** - Patterns and approaches for bringing new players in.

These guides are informed by real usage patterns across the platform and are designed to help you avoid common pitfalls while getting the most out of Jest.

If you're looking for API details or implementation specifics, see the SDK documentation.

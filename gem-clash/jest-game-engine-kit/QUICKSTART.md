# Quick Start Guide

## 1. Open the Engine
Open `jest-game-engine.html` in your browser (Chrome recommended).

## 2. Name Your Project
Double-click "Untitled Game" in the header → Type your game name → Press Enter.

## 3. Create Screens
Click **"+ Add Screen"** to add a new screen. Common screens:
- Main Menu
- Gameplay
- Shop
- Settings
- Level Select

Double-click a tab to rename it.

## 4. Add Components

### Add a Button
1. Click **"+ Add Button"** in the Buttons section
2. Choose a shape (Circle, Square, Rectangle, Oval)
3. Drag it on the preview to position
4. Drag corner handles to resize
5. Enter trigger behavior (e.g., "On click: start game")

### Add Text
1. Click **"+ Add Text"** in the Text section
2. Enter your text content
3. Adjust font size and color
4. Optional: Upload a custom font file

### Add a Counter
1. Click **"+ Add Counter"** in the Counters section
2. Set initial value (e.g., 0)
3. Set format (e.g., "Score: {value}")
4. Describe behavior (e.g., "Increases by 10 per match")

### Add a Game Board
1. Click **"+ Add Game Board"** in the Game Board section
2. Set rows and columns
3. Adjust cell size
4. Describe mechanics (e.g., "Match-3 puzzle gameplay")

## 5. Upload Assets
1. Click **"Upload Assets"** in Asset Library
2. Select PNG/JPG/SVG files
3. Click an asset while a button is selected to apply it

**Tip**: Black backgrounds (#000000) become transparent automatically.

## 6. Customize Colors
- Change default colors in the Colors section
- Add custom colors by typing in the prompt window:
  - "add a glow effect color"
  - "add a health bar color"

## 7. Switch Screens
Click tabs at the top of the preview to switch screens. Each screen keeps its own elements.

## 8. Save & Export
- Click **"Save"** to save to browser storage
- Click **"Export JSON"** to download the project file

## Tips
- Hold and drag elements to move them
- Drag corner handles to resize
- Double-click component labels to rename
- Click outside elements to deselect
- Use the Properties panel (right) for precise positioning
